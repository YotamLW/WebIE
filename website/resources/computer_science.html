Numbering Systems

Denary / decimal (numbering system of order 10) is the standard human numbering system which has ten digits (0 through 9). To achieve higher numbers than there are digits, bases are used, which are shown as adjacent digits. Each base to the left of another base represents a full "digits-worth" of that other base. For example, in denary there are ten digits, but there is no single digit for ten, so instead the base to the left is incremented thus ten is written as 10 (which shows 1 $\times$ number of digits + 0). This is called an overflow - the number is flowing over to the next base. Both the digits and the numbering system can be changed depending on use case. For example, base 16 (hexadecimal) has 16 digits - it has 0 through 9, but then the letter A to represent ten, the letter B to represent eleven and so on (so in this base (numbering system) 10 would be sixteen). To make things easier to write, when working with different bases a subscript of their base (in denary) is included, so sixteen in hexadecimal would be written as 10$_{16}$ and sixteen in denary would be written as 16$_{10}$. Binary is the numbering system of base 2, using the digits 0 and 1. This is used in computers mainly because it is easier to understand binary logic and it is easier and cheaper to make hardware for. Mathematical logic involving binary is called boolean logic, where the digit 1 represents true and the digit 0 represents false.
